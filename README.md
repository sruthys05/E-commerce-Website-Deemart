# E-commerce Website (DeeMart)

This is a full-stack e-commerce web application built using React.js for frontend and Spring Boot for backend.

## Features
- Product listing and search
- Shopping cart functionality
- User authentication
- Order management
- Responsive UI for all devices

## Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- API: RESTful services

## How to run
Frontend:
npm install
npm start

Backend:
Run Spring Boot application from IDE
